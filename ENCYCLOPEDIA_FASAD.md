# РАЗДЕЛ 2. НВФ — НАВЕСНЫЕ ВЕНТИЛИРУЕМЫЕ ФАСАДЫ

## 2.1 Принцип и состав системы

**Принцип:** облицовка крепится к несущей конструкции через подсистему (каркас), между облицовкой и утеплителем — вентилируемый зазор 40-60 мм.

**Слои НВФ (снаружи внутрь):**
```
1. Облицовка (керамогранит, HPL, металлокассеты, фиброцемент, натуральный камень)
2. Вентилируемый зазор (40-60 мм)
3. Ветрозащитная мембрана (паропроницаемая)
4. Утеплитель минеральная вата (100-200 мм)
5. Несущая стена (монолит, кирпич, газобетон)
```

**Запрещено** использовать пенопласт (ПСБ) в НВФ — пожарные нормы!

### Толщина утеплителя (Москва):
- Минимум по теплозащите R=3.28 м²·°С/Вт (стена жилого дома)
- Минвата λ=0.036: h = 3.28 × 0.036 = 118 мм → **принять 150-200 мм**
- Двухслойная укладка (100+50 или 150+50 мм) с разбежкой швов

### Ветрозащитная мембрана
- Паропроницаемая, ветростойкая
- Марки: Изоспан А, Tyvek, Jutadach 115
- Цена: 20-60 руб/м²
- Крепление: тарельчатые дюбеля (10-12 шт/м²)

# РАЗДЕЛ 3. МОКРЫЕ ФАСАДЫ (СФТК)

## 3.1 Система фасадная теплоизоляционная композиционная (СФТК)

**Слои СФТК (снаружи внутрь):**
```
1. Финишный слой (декоративная штукатурка или краска)
2. Затирочный слой (армирующая сетка + тонкий базовый штукатурный слой)
3. Клеевой слой (минеральный клей)
4. Утеплитель (пенополистирол ПСБ-С-25Ф или минвата)
5. Клеевой слой (первичная приклейка)
6. Несущая стена
```

# РАЗДЕЛ 11. ШПАРГАЛКИ ДЛЯ АГЕНТОВ

## 11.1 Быстрый расчёт площади фасада
```
S_фасада = периметр здания × высота этажей
Вычесть: площадь оконных проёмов (~20-40% от площади фасада)
Добавить: торцевые стены, парапеты, цоколь

Соотношение: остекление / глухой фасад для ЖК:
- Эконом: 25-35% остекление
- Комфорт: 30-45%
- Бизнес: 40-60%
```

## 11.2 Контрольные соотношения

```
Леса ≈ 15-25% от стоимости фасадных работ (отдельная позиция)
Утеплитель ≈ 20-30% от стоимости НВФ «под ключ»
Подсистема ≈ 15-25% от стоимости НВФ
Облицовка ≈ 30-50% от стоимости НВФ
Монтажные работы ≈ 25-40% от стоимости НВФ

Для СФТК:
Утеплитель + клей ≈ 40-50%
Армирование + штукатурка ≈ 30-40%
Работы ≈ 40-50% от итога
```

## 11.3 Типичная структура ВОР фасада
```
1. Устройство лесов (аренда + монтаж/демонтаж)
2. Огрунтовка основания
3. [НВФ]: монтаж кронштейнов и подсистемы
   [СФТК]: нанесение клея
4. Утеплитель (приклейка + дюбелирование)
5. [НВФ]: ветрозащитная мембрана
   [СФТК]: армирующий слой (сетка + базовый штукатурный)
6. [НВФ]: облицовка
   [СФТК]: декоративная штукатурка
7. Герметизация швов (оконные примыкания)
8. Отливы оконные
9. Демонтаж лесов
```

# РАЗДЕЛ 13. ТИПИЧНЫЕ ОШИБКИ ПРИ РАСЦЕНКЕ ФАСАДОВ

## 13.1 Критические ошибки

**Ошибка 1: Площадь фасада ≠ площадь стены в плане**
- Фасадная площадь учитывает высоту этажей × периметр, МИНУС проёмы
- Стандарт: проёмы вычитаются, если площадь проёма >0.5 м²
- Углы, пилястры, эркеры — считаются по развёртке

**Ошибка 2: Не разграничены СФТК и НВФ**
- СФТК (штукатурный) и НВФ — принципиально разные системы с разной ценой
- СФТК дешевле (~2 500–4 500 руб/м²), НВФ дороже (~4 000–12 000 руб/м²)

**Ошибка 3: Не учтено утепление откосов**
- Откосы оконных и дверных проёмов требуют отдельного утепления
- Обычно 50 мм ЭППС + штукатурка: 400–800 руб/пм

**Ошибка 4: Дюбели отдельной строкой**
- Иногда дюбели включены в расценку, иногда — нет
- Проверить: 5–8 шт/м² × 15–25 руб/шт = 75–200 руб/м² отдельно

**Ошибка 5: Нет надбавки за высоту лесов**
- Леса — отдельная позиция или % к фасадным работам
- Леса рамные аренда: 200–400 руб/м²/мес
- Леса рамные монтаж+демонтаж: 300–600 руб/м²

## 13.2 Частые недочёты

- Нет армирования углов здания (уголок ПВХ + сетка) — 150–300 руб/пм
- Не учтён водозащитный профиль у основания системы
- Декоративная штукатурка: цвет ≠ белый → колеровка +50–150 руб/м²
- НВФ: не учтены доборные элементы (оконные откосы, отливы, углы)

# РАЗДЕЛ 20. ШПАРГАЛКА: БЫСТРАЯ РАСЦЕНКА ФАСАДОВ

## СФТК — алгоритм за 2 минуты

1. **Площадь**: периметр × высота − проёмы (площадь проёма >0.5 м²)
2. **Утеплитель**: толщина по теплорасчёту (обычно 80–150 мм)
   - 80 мм минвата + 10 мм штукатурный слой → итого 90 мм
3. **Материалы**: клей + утеплитель + дюбели + сетка + грунт + декоративная штукатурка
4. **Откосы**: периметр всех проёмов × 0.3–0.5 м (ширина откоса)
5. **Проверка цены**:
   - Просто штукатурка без утепления: 800–1 400 руб/м²
   - СФТК 80 мм минвата + декор: 2 500–4 500 руб/м²
   - СФТК 100 мм + силиконовая штукатурка: 3 500–6 500 руб/м²

## НВФ — алгоритм за 2 минуты

1. **Площадь**: та же (периметр × высота − проёмы >0.5 м²)
2. **Подсистема**: алюминий (стандарт) / нержавеющая сталь (агрессивная среда)
3. **Утеплитель**: 50–100 мм + ветрозащита
4. **Воздушный зазор**: 30–60 мм (обязательно!)
5. **Облицовка**: фиброцемент / керамогранит / АКП / клинкер
6. **Проверка цены**:
   - Фиброцемент 1200–1500 мм: 4 000–7 000 руб/м²
   - Керамогранит 600×600: 6 000–12 000 руб/м²
   - АКП 3 мм: 5 000–9 000 руб/м²

# ЛОВУШКИ ЕДИНИЦ ИЗМЕРЕНИЯ — ЧИТАЙ ПЕРЕД РАСЦЕНКОЙ!

### Правило 1: Площадь фасада != площадь стены в плане
```
ОШИБКА: считать площадь фасада как «периметр × высота» БЕЗ вычета проёмов.
ПРАВИЛЬНО: площадь = (периметр × высота) − площадь проёмов (>0.5 м2)

Типичное соотношение:
- Эконом-класс: остекление 25-35% → вычитаем 25-35%
- Комфорт-класс: остекление 30-45% → вычитаем 30-45%
- Бизнес-класс: остекление 40-60% → вычитаем 40-60%
```

### Правило 2: СФТК vs. НВФ — разные системы, разные цены
```
КРИТИЧЕСКАЯ ОШИБКА: применить цену СФТК к позиции НВФ или наоборот.
СФТК (штукатурный): 2,500-7,600 руб/м2
НВФ (вентилируемый): 4,000-22,000 руб/м2

Разница может быть в 2-3 раза! Определяй тип по ключевым словам:
СФТК → штукатурка, декоративная, короед, барашек, армирующий слой, клеевой слой
НВФ → подсистема, кронштейн, вентзазор, клямера, кассеты
```

### Правило 3: Леса — ОТДЕЛЬНАЯ позиция, всегда!
```
Леса = 15-25% от стоимости фасадных работ.
В ВОР может быть:
а) Включены в комплекс → цена выше
б) Отдельной строкой: «Аренда + монтаж/демонтаж лесов» → 500-1,200 руб/м2

ОШИБКА: забыть добавить леса → занижение на 15-25%.
ОШИБКА: двойной учёт лесов (и в комплексе, и отдельно) → завышение на 15-25%.
```

### Правило 4: Откосы — отдельная позиция
```
Откосы (оконные и дверные) — не входят в стоимость фасада!
Утепление откоса: 400-800 руб/пм
Количество: периметр всех проёмов × ширина откоса (0.2-0.5 м)

Для 100 окон 1.5×1.5 м: периметр = 100 × 6 = 600 пм
Стоимость: 600 × 600 = 360,000 руб → существенная статья!
```

### Правило 5: Единица м2 для фасада — всегда по развёртке
```
М2 для фасада = по РАЗВЁРТКЕ наружной поверхности.
Эркеры, пилястры, балконы → считать все грани!
Лоджии → если утепляются/штукатурятся → боковые + задняя стена.
```

## 28.4 Нормативные требования

- **СП 59.13330** — требования к безбарьерной среде (СОФ для инвалидов не предусмотрена, но учитывать доступ)
- **ГОСТ Р 58962-2020** — системы обслуживания фасадов, требования безопасности
- **ПОТ Р М-012-2000** — правила работы на высоте
- BMU обязательна для зданий с витражным фасадом >75 м (мойка, ремонт)
- Анкерные линии обязательны для ЛЮБОГО здания с обслуживаемым фасадом

## 28.5 Типичные ошибки

- **СОФ отсутствует в ВОР:** для высотных зданий обязательна — без неё фасад нельзя обслуживать
- **Только анкеры без линий:** анкерные точки без тросовой системы — работать неудобно и опасно
- **Нет испытаний:** все СОФ обязательно проходят статические и динамические испытания
- **BMU не по проекту:** BMU проектируется под конкретное здание, типовых решений мало
