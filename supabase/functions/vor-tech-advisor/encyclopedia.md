# РАЗДЕЛ 2. НВФ — состав, подсистемы, облицовки, утеплитель

## 2.1 Принцип и состав системы

**Принцип:** облицовка крепится к несущей конструкции через подсистему (каркас), между облицовкой и утеплителем — вентилируемый зазор 40-60 мм.

**Слои НВФ (снаружи внутрь):**
```
1. Облицовка (керамогранит, HPL, металлокассеты, фиброцемент, натуральный камень)
2. Вентилируемый зазор (40-60 мм)
3. Ветрозащитная мембрана (паропроницаемая)
4. Утеплитель минеральная вата (100-200 мм)
5. Несущая стена (монолит, кирпич, газобетон)
```

## 2.2 Подсистемы (каркас НВФ)

### Алюминиевые подсистемы (ALUTECH, Краспан, РОНСОН)
| Тип | Применение |
|-----|------------|
| Т-образный профиль + кронштейны | Стандарт для ЖК |
| Омега-профиль | Лёгкая облицовка |
| Несущий паз (для стекла) | Стекло, HPL |

### Стальные оцинкованные подсистемы (бюджетный сегмент)
| Тип | Применение |
|-----|------------|
| Г-образный кронштейн + несущий профиль | Бюджетные ЖК |
| Перфорированный С-профиль | Лёгкие облицовки |

### Крепёж
- **Анкерный дюбель** в монолит: 8-14 мм, глубина 60-120 мм
- **Расход дюбелей:** 4-8 шт/м² (в зависимости от ветровой нагрузки)

## 2.4 Утеплитель НВФ

### Минеральная вата (обязательно негорючая — НГ или Г1)
| Марка | Плотность, кг/м³ | λ, Вт/(м·К) | Примечание |
|-------|------------------|-------------|------------|
| ROCKWOOL Фасад Баттс | 80 | 0.036 | Стандарт для НВФ |
| ROCKWOOL Фасад Баттс Д | 90 | 0.036 | Повышенная плотность |
| Paroc FAS 3 | 80-100 | 0.034 | Финский |
| ISOVER Фасад | 80 | 0.036 | |
| ТехноФас | 80-115 | 0.036 | |

**Запрещено** использовать пенопласт (ПСБ) в НВФ — пожарные нормы!

### Толщина утеплителя (Москва):
- Минимум по теплозащите R=3.28 м²·°С/Вт (стена жилого дома)
- Минвата λ=0.036: h = 3.28 × 0.036 = 118 мм → **принять 150-200 мм**
- Двухслойная укладка (100+50 или 150+50 мм) с разбежкой швов

### Ветрозащитная мембрана
- Паропроницаемая, ветростойкая
- Марки: Изоспан А, Tyvek, Jutadach 115
- Крепление: тарельчатые дюбеля (10-12 шт/м²)

# РАЗДЕЛ 3. Мокрые фасады (СФТК)

## 3.1 Система фасадная теплоизоляционная композиционная (СФТК)

**Слои СФТК (снаружи внутрь):**
```
1. Финишный слой (декоративная штукатурка или краска)
2. Затирочный слой (армирующая сетка + тонкий базовый штукатурный слой)
3. Клеевой слой (минеральный клей)
4. Утеплитель (пенополистирол ПСБ-С-25Ф или минвата)
5. Клеевой слой (первичная приклейка)
6. Несущая стена
```

# РАЗДЕЛ 10. Шпаргалки

## 10.1 Быстрый расчёт площади фасада
```
S_фасада = периметр здания × высота этажей
Вычесть: площадь оконных проёмов (~20-40% от площади фасада)
Добавить: торцевые стены, парапеты, цоколь

Соотношение: остекление / глухой фасад для ЖК:
- Эконом: 25-35% остекление
- Комфорт: 30-45%
- Бизнес: 40-60%
```

## 10.2 Контрольные соотношения

```
Леса ≈ 15-25% от стоимостной части фасадных работ (отдельная позиция)
Утеплитель ≈ 20-30% от итоговой стоимости НВФ «под ключ»
Подсистема ≈ 15-25% от итоговой стоимости НВФ
Облицовка ≈ 30-50% от итоговой стоимости НВФ
Монтажные работы ≈ 25-40% от итоговой стоимости НВФ

Для СФТК:
Утеплитель + клей ≈ 40-50%
Армирование + штукатурка ≈ 30-40%
Работы ≈ 40-50% от итога
```

## 10.3 Типичная структура ВОР фасада
```
1. Устройство лесов (аренда + монтаж/демонтаж)
2. Огрунтовка основания
3. [НВФ]: монтаж кронштейнов и подсистемы
   [СФТК]: нанесение клея
4. Утеплитель (приклейка + дюбелирование)
5. [НВФ]: ветрозащитная мембрана
   [СФТК]: армирующий слой (сетка + базовый штукатурный)
6. [НВФ]: облицовка
   [СФТК]: декоративная штукатурка
7. Герметизация швов (оконные примыкания)
8. Отливы оконные
9. Демонтаж лесов
```

# РАЗДЕЛ 13. Типичные ошибки

## 13.1 Критические ошибки

**Ошибка 1: Площадь фасада ≠ площадь стены в плане**
- Фасадная площадь учитывает высоту этажей × периметр, МИНУС проёмы
- Стандарт: проёмы вычитаются, если площадь проёма >0.5 м²
- Углы, пилястры, эркеры — считаются по развёртке

**Ошибка 2: Не разграничены СФТК и НВФ**
- СФТК (штукатурный) и НВФ — принципиально разные системы с разной ценой
- НВФ существенно дороже СФТК — разница может быть в 2–3 раза
- Определять тип нужно по ключевым словам: СФТК → штукатурка, декоративная, короед, барашек, армирующий слой; НВФ → подсистема, кронштейн, вентзазор, клямера, кассеты

**Ошибка 3: Не учтено утепление откосов**
- Откосы оконных и дверных проёмов требуют отдельного утепления
- Обычно 50 мм ЭППС + штукатурка — отдельная позиция к ВОР (по пм)

**Ошибка 4: Дюбели отдельной строкой**
- Иногда дюбели включены в расценку, иногда — нет
- Проверить: 5–8 шт/м² × норма цены за штуку — это отдельная статья к ВОР

**Ошибка 5: Нет надбавки за высоту лесов**
- Леса — отдельная позиция или процент к фасадным работам
- Леса рамные: аренда отдельно, монтаж и демонтаж отдельно

## 13.2 Частые недочёты

- Нет армирования углов здания (уголок ПВХ + сетка) — отдельная позиция по пм
- Не учтён водозащитный профиль у основания системы
- Декоративная штукатурка: цвет ≠ белый → колеровка добавляет надбавку
- НВФ: не учтены доборные элементы (оконные откосы, отливы, углы)

## 13.3 Ловушки единиц измерения

### Правило 1: Площадь фасада ≠ площадь стены в плане
```
ОШИБКА: считать площадь фасада как «периметр × высота» БЕЗ вычета проёмов.
ПРАВИЛЬНО: площадь = (периметр × высота) − площадь проёмов (>0.5 м²)
```

### Правило 2: СФТК vs. НВФ — разные системы
```
КРИТИЧЕСКАЯ ОШИБКА: применить расценку СФТК к позиции НВФ или наоборот.
Разница в комплексной стоимости — в 2–3 раза.
```

### Правило 3: Леса — ОТДЕЛЬНАЯ позиция, всегда!
```
В ВОР возможны два варианта:
а) Леса включены в комплексную расценку фасада
б) Леса отдельной строкой: «Аренда + монтаж/демонтаж лесов»

ОШИБКА: забыть добавить леса → существенное занижение.
ОШИБКА: двойной учёт лесов (и в комплексе, и отдельно) → завышение.
```

### Правило 4: Откосы — отдельная позиция
```
Откосы (оконные и дверные) — не входят в стоимость основного фасада!
Утепление откоса считается по пм.
Количество: периметр всех проёмов × ширина откоса (0.2–0.5 м).
```

### Правило 5: Единица м² для фасада — всегда по развёртке
```
М² для фасада = по РАЗВЁРТКЕ наружной поверхности.
Эркеры, пилястры, балконы → считать все грани!
Лоджии → если утепляются/штукатурятся → боковые + задняя стена.
```

# РАЗДЕЛ 14. Технологические цепочки и обязательные спутники

Каждый фасадный материал или работа тянет за собой набор обязательных «спутников» — без них система либо не работает технически, либо нарушает нормы. При проверке ВОР рядом с главной позицией всегда должны присутствовать спутники. Если их нет — это либо забыли, либо «спрятали» в составе главной позиции.

| Главный материал/работа | Обязательные спутники | Обоснование |
|---|---|---|
| Утеплитель НВФ (минвата) | Ветрозащитная паропроницаемая мембрана + тарельчатые дюбели + подсистема (кронштейны и направляющие) | Без мембраны утеплитель мокнет от косого дождя, без дюбелей не держится на стене, без подсистемы нет вентзазора |
| Утеплитель СФТК (минвата/ПСБ) | Клей для приклейки + тарельчатые дюбели + базовый армирующий слой со стеклотканевой сеткой | Утеплитель без армирования трескается под финишем; без дюбелей отрывается от стены при ветровой нагрузке |
| Утеплитель СФТК с ПСБ при h>28 м | Замена на минвату НГ (запрещено ПСБ выше 28 м) + противопожарные рассечки из минваты h=200 мм на каждом перекрытии | ФЗ-123 и СП 112.13330: горючий утеплитель на высотном здании — нарушение пожарных норм |
| Клинкерная плитка (НВФ или мокрая укладка) | Затирка швов (FM-смеси) + расшивка + торцевые/угловые элементы | Без расшивки швы протекают, видны металлические элементы крепления; без угловых элементов углы здания «рваные» |
| Натуральный камень (НВФ) | Усиленная подсистема (нержавейка) + индивидуальные анкеры/штифты + герметик швов | Камень тяжёлый (50–80 кг/м²), стандартные кляммеры не держат; швы герметизируются силиконом |
| Облицовочный кирпич (мокрая кладка) | Гибкие связи к монолиту (4–8 шт/м²) + цементно-песчаный раствор М100–М150 + затирка/фугование швов + опорные полки на перекрытиях | Кирпич без связей отваливается от несущей стены; без фугования швы выкрашиваются |
| Металлокассеты / АКП (НВФ) | Подсистема + заклёпки вытяжные (10–15 шт/м²) + закладные элементы для фиксации + торцевые планки | Кассеты не клипсуются как керамогранит — их клепают; нужны торцевые планки для краёв |
| Керамогранит крупного формата (НВФ) | Спецкляммеры под формат + усиленная подсистема + механический крепёж в дополнение к кляммерам | Большой формат имеет большой вес и парусность; стандартные кляммеры не держат |
| Фиброцементные плиты (НВФ) | Подсистема + ветрозащитная мембрана + торцевые/стартовые/финишные планки + заклёпки | Плиты крепятся видимым крепежом, без планок край неаккуратный и пропускает воду |
| Стеклянные ограждения балконов | Стойки нержавейка + закалённое стекло или триплекс (НЕ обычное!) + поручень + герметик швов | По ГОСТ 30698 обычное стекло запрещено для ограждений — только закалённое или триплекс; без поручня стекло хлипкое |
| Козырёк входной (стеклянный на тягах) | Тяги/штанги нержавейка + закалённое стекло + анкерная заделка в монолит + водоотведение | Без надёжной заделки тяг козырёк падает; без водоотведения вода попадает на двери |
| Козырёк входной (металлический) | Стальной каркас + профнастил/поликарбонат + водоотведение (желоб + воронка) + покраска/оцинковка | Без водоотведения капель на голову входящим; без оцинковки каркас ржавеет |
| Откосы оконные/дверные | Утепление откоса (50 мм ЭППС/минвата) + штукатурка по сетке ИЛИ панель + ПСУЛ-лента + герметик примыкания | Без утепления откоса — мостик холода и конденсат; без ПСУЛ — продувание оконного шва |
| Отливы оконные | Кронштейны/закладные + герметизация бокового примыкания + капельник на вылете + подкладочная лента | Без капельника вода течёт по стене; без герметизации боков — затёки в шов |
| Оштукатуривание (СФТК или мокрый фасад) | Грунтовка основания + армирующая стеклотканевая сетка 145 г/м² + грунтовка под финиш + декоративная штукатурка | Без грунтовки штукатурка отслаивается; без сетки трескается на стыках утеплителя |
| Противопожарные отсечки (рассечки) из минваты | Установка на каждом перекрытии при ПСБ + полное прохождение всей толщины утеплителя + стыковка с мембраной | По СП 112: горизонтальная рассечка h=200 мм на каждом этаже при горючем утеплителе |
| Витраж стоечно-ригельный | Стоечный профиль + ригельный профиль + стеклопакет + уплотнители EPDM + прижимная планка + декоративная крышка + герметик примыкания к монолиту | Без уплотнителей — продувание; без герметика примыкания — затёки между витражом и стеной |
| Витраж/окно — оконный блок | Подоконник + отлив + откосы + ПСУЛ-лента + монтажная пена + пароизоляция шва (по ГОСТ 30971 — три слоя) | По ГОСТ 30971 монтажный шов состоит из трёх слоёв: пена + ПСУЛ снаружи + пароизоляция изнутри |
| Тамбур (входная группа) | Алюминиевый профиль + стеклопакеты + 2+ двери + порог (гранит/бетон) + доводчики + уплотнители + герметизация примыкания | Без порога — пыль и вода в подъезде; без доводчиков двери хлопают; без герметизации — продувание |
| Наружная дверь (алюминиевая или стальная) | Замок + ручка + доводчик + порог + монтажный шов (пена+ПСУЛ+пароизоляция) + утеплённые откосы | Без доводчика — хлопают и не закрываются; ГОСТ 30971 требует трёхслойный шов |
| Противопожарная дверь EI 60 | Сертификат пожарной безопасности + специальная фурнитура (антипаник) + доводчик + сертифицированный монтаж с актом | По ФЗ-123 без сертификата дверь нельзя ставить на эвакуационном выходе |
| Леса фасадные | Аренда (в месяцах) + монтаж + демонтаж + защитная сетка (по ТБ) + защитный козырёк над пешеходными зонами + проектирование при h>40 м | По ПОТ-Р леса без сетки и козырька — нарушение охраны труда; на высотных нужен ППР |
| СОФ (система обслуживания фасадов) | Анкерная линия по парапету + анкерные точки + проектирование под здание + статические/динамические испытания + сертификация | Для зданий с витражным фасадом >75 м обязательна BMU; без испытаний СОФ нельзя ввести в эксплуатацию |
| Балконные ограждения | Стойки + заполнение (решётка/стекло) + поручень + крепление в плиту перекрытия (анкеры) + защита от коррозии | По СП 54.13330 высота ≥1200 мм обязательна; крепление в плиту, не в облицовку |
| Кровельное ограждение (парапетное) | Стойки + горизонтальные элементы + крепление в парапет + ограничитель снега (если есть) | По СП 17.13330 обязательно для всех плоских кровель; высота ≥600 мм при наличии парапета, ≥1200 мм без него |
