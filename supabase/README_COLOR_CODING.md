# Цветовая кодировка задач по ответственным

## Описание функционала

Теперь каждый член команды имеет свой цвет, который визуально выделяет назначенные на него задачи:
- **Вячеслав Зинин** - синий (#3b82f6)
- **Валерий Коваленко** - фиолетовый (#8b5cf6)
- **Никита Кузнецов** - зеленый (#10b981)

Задачи визуально выделяются:
- Цветная левая граница (4px)
- Полупрозрачный фон цвета ответственного
- Цветная индикация в селекте ответственного

## Миграции

### 011_add_color_to_team_members.sql
Добавляет поле `color` в таблицу `team_members` и устанавливает цвета по умолчанию для существующих членов команды.

## Изменения в коде

### AdminPage.jsx
- Добавлено поле `color` в форму добавления/редактирования члена команды
- Color picker для выбора цвета (HTML5 input type="color")
- Сохранение цвета в БД при создании/обновлении

### AdminPage.css
- Стили для color picker
- Форма с полем выбора цвета

### TasksPage.jsx
- Загрузка поля `color` из team_members
- Визуальное выделение строк задач:
  - `borderLeft: 4px solid ${color}` - цветная левая граница
  - `backgroundColor: ${color}08` - полупрозрачный фон (08 = 8% прозрачности)
- Индикация цвета в селекте ответственного:
  - Цветная левая граница у селекта
  - Цветные точки (●) рядом с именами в опциях

## Как применить

### 1. Выполните миграцию в Supabase

```sql
-- Скопируйте и выполните содержимое файла:
-- 011_add_color_to_team_members.sql
```

### 2. Код уже обновлен

Все изменения в коде уже применены:
- ✅ AdminPage.jsx
- ✅ AdminPage.css
- ✅ TasksPage.jsx

### 3. Проверка

1. Откройте админ-панель → Члены команды
2. Убедитесь, что у каждого члена есть выбор цвета
3. Откройте задачи любого объекта
4. Назначьте ответственных на задачи
5. Убедитесь, что задачи визуально выделяются цветом ответственного

## Примеры использования

### В админ-панели
- При создании нового члена команды выберите цвет
- При редактировании можно изменить цвет
- Цвет отображается рядом с номером порядка

### На странице задач
- Задачи с назначенными ответственными подсвечиваются их цветом
- В селекте ответственного каждый член показан со своей цветной точкой
- Легко найти свои задачи по цвету
