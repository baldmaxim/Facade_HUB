# РАЗДЕЛ 11. ПИРОГ СФТК — СОСТАВ И НОРМЫ РАСХОДА

## 11.1 Стандартный пирог СФТК (снаружи→внутрь)

```
1. Финишный декоративный слой (штукатурка "короед", "барашек")
2. Грунтовка кварцевая (адгезия под финиш)
3. Базовый армирующий слой (клей-армировка)
4. Стеклотканевая сетка 145 г/м² (утоплена в базовый слой)
5. Утеплитель (минвата / ЭППС) на клей + дюбели
6. Грунтовка (адгезионная, по основанию)
7. Основание (бетон/кирпич/газобетон)
```

## 11.3 Пирог НВФ (снаружи→внутрь)

```
1. Облицовка (керамогранит/фиброцемент/АКП)

3. Ветрозащитная паропроницаемая мембрана
4. Утеплитель двухслойный (нижний мягкий + верхний жёсткий) на кронштейнах
```

# РАЗДЕЛ 13. ТИПИЧНЫЕ ОШИБКИ ПРИ РАСЦЕНКЕ ФАСАДОВ

## 13.1 Критические ошибки

**Ошибка 1: Площадь фасада ≠ площадь стены в плане**
- Фасадная площадь учитывает высоту этажей × периметр, МИНУС проёмы
- Стандарт: проёмы вычитаются, если площадь проёма >0.5 м²
- Углы, пилястры, эркеры — считаются по развёртке

**Ошибка 2: Не разграничены СФТК и НВФ**
- СФТК (штукатурный) и НВФ — принципиально разные системы с разной ценой
- СФТК дешевле (~2 500–4 500 руб/м²), НВФ дороже (~4 000–12 000 руб/м²)

**Ошибка 3: Не учтено утепление откосов**
- Откосы оконных и дверных проёмов требуют отдельного утепления
- Обычно 50 мм ЭППС + штукатурка: 400–800 руб/пм

**Ошибка 4: Дюбели отдельной строкой**
- Иногда дюбели включены в расценку, иногда — нет
- Проверить: 5–8 шт/м² × 15–25 руб/шт = 75–200 руб/м² отдельно

**Ошибка 5: Нет надбавки за высоту лесов**
- Леса — отдельная позиция или % к фасадным работам
- Леса рамные аренда: 200–400 руб/м²/мес
- Леса рамные монтаж+демонтаж: 300–600 руб/м²

## 13.2 Частые недочёты

- Нет армирования углов здания (уголок ПВХ + сетка) — 150–300 руб/пм
- Не учтён водозащитный профиль у основания системы
- Декоративная штукатурка: цвет ≠ белый → колеровка +50–150 руб/м²
- НВФ: не учтены доборные элементы (оконные откосы, отливы, углы)

# РАЗДЕЛ 15. ЧЕКЛИСТ ПРОВЕРКИ ФАСАДНОЙ ПОЗИЦИИ

```
□ Тип системы: СФТК / НВФ / штукатурка без утепления / облицовка?
□ Площадь правильно считается? (по развёртке, минус проёмы >0.5 м²)
□ Утеплитель: тип, толщина, плотность?
□ Клей + дюбели: есть в расценке или отдельно?
□ Сетка армирующая: учтена?
□ Декоративная штукатурка: тип (короед/барашек/мозаика), зерно?
□ Откосы проёмов: отдельная позиция?
□ Углы здания: армирование учтено?
□ Леса: аренда + монтаж/демонтаж предусмотрены?
□ Высота: коэффициент учтён?
□ Для НВФ: подсистема + утеплитель + ветрозащита + облицовка — все позиции?
□ Единица: м² (правильная)?
```
